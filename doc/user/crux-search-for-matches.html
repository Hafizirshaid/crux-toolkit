crux search-for-matches

Description:
This command searches a protein database with a set of spectra. For each spectrum, candidate peptides are identified whose inferred m/z lies within a specified range of the observed precursor m/z. These candidate peptides are first ranked with the SEQUEST Sp score. The top 500 matches are then re-scored using XCorr. The top 5 peptides for each spectrum are output by the program. The input protein database may either be in FASTA format or it may be a binary index created by cvs create-index.
Usage:
crux search-for-matches [options] <ms2 input filename> <protein input>

Input:

  • <ms2 > – The name of the file (in MS2 format) from which to parse the spectra.
  • <protein input> – The name of the file in fasta format or the directory containing a protein index from which to retrieve proteins and peptides.
Output:
By default, the program produces a binary .csm file containing one
Options:
  • [--output-mode <sqt | binary | all> ] – The output format. Default = binary.
  • [--match-output-folder <string>] – The output folder to store the serialized binary output files. Default = current directory.
  • [--sqt-output-file <string>] – Name of the sqt file to place matches. Default = target.sqt.
  • [--decoy-sqt-output-file <string>] – Name of the sqt file to place matches. Default = decoy.sqt.
  • [--spectrum-min-mass <float>] – The lowest spectrum m/z to search in the ms2 file. Default = 0.0
  • [--spectrum-max-mass <float>] – The highest spectrum m/z to search in the ms2 file. Default = Positive Infinity.
  • [--prelim-score-type <sp>] – The type of preliminary scoring function to use. Default = sp.
  • [--score-type <xcorr | sp | dotp| xcor_logp | sp_logp>] – The type of scoring function to use. Default = xcorr.
  • [--number-decoy-set <int>] – Specify the number of decoy sets to generate for match_analysis. Default = 2.
  • [--spectrum-charge <1 | 2 | 3 | all>] – The spectrum charges to search. Default = all.
  • [--overwrite <T|F>] Replace existing files if true (T) or fail when trying to overwrite a file if false (F). Default = F.
  • [--use-index <T|F>] Use an index that has already been created if true (T) or use a fasta file if false (F). Default = F.
  • [--parameter-file <filename>] – A file containing command-line or additional parameters. See the parameter documentation page for details.
  • [--verbosity <int>] – Specify the verbosity of the current processes from 0-100. Default = 40.
Warning messages:
  • Fatal: The file "spectra input filename" does not exist (or is not readable, or is empty).
  • Fatal: The file "protein input filename" does not exist (or is not readable, or is empty).
  • Fatal: The following options are incompatible:
    • --output-mode = binary and --sqt-output-file
